    # 配置参数（Attributes）继承 avue-form

    参数 说明 类型 默认值
    option 配置项（使用时加上.sync 修饰符） Object 无
    params 绑定值 Object 无
    option.url 搜索接口 url String 无
    tableLoading avue-crud 的 loading 值 （使用时加上.sync 修饰符） Boolean 无
    tableData avue-crud 的 data 数据（使用时加上.sync 修饰符） Array 无
    keyVal 修改默认键值对，目前只支持对后端返回列表数据 records 修改 Object { list: "records" }

    # 事件（Event）继承 avue-form

    参数 说明 类型 返回值示例
    search 搜索事件触发，返回接口调用后的结果 Function {code:200,data:{},msg:'',success:false}
